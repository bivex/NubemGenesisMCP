## File Format
Each persona file follows a consistent YAML structure:
- name: Persona identifier
- level: L4 (Senior/Expert level)
- category: One of 7 categories
- version: 1.0.0
- identity: Description of the persona
- specialties: List of specialized skills
- capabilities: List of key capabilities
- system_prompt: Detailed prompt (200-300+ lines)
- confidence_scores: Skill confidence ratings
- collaborates_with: Related personas
- keywords: Searchable keywords
- metrics: Performance metrics
- commands: Available commands
